Huffman Coding

Greedy algorithm used for data compression by assigning shorter codes to frequent characters.

  • Count frequency of each character
  • Build binary tree using lowest frequencies
  • Assign prefix-free binary codes

Time: O(n log n)

Property: No code is prefix of another
